# 🌐 University Chat for Software Engineering

Welcome to the **University Chat for Software Engineering** — a real-time academic forum project built with Firebase and deployed on GitHub Pages.

---

## 🧠 About the Project

This is a **web-based social platform** inspired by Discord and Reddit, created as a personal project by a Software Engineering student.

The system includes:

- 🔐 Google Authentication (Firebase Auth)
- 👤 User profiles (username + avatar)
- 💬 Real-time posts (Firestore database)
- 🧵 Comments system (future feature)
- 🌍 Multiple internal servers (like Discord channels)
- ⚡ Live updates using Firebase onSnapshot

---

## 🚀 Live Features

### 🔐 Authentication
- Login with Google account
- Secure session handling
- Profile creation after login

### 👤 User System
- Custom username
- Profile picture from Google
- Optional security password (stored in Firestore)

### 💬 Forum System
- Real-time posts
- Server-based separation (General / AI / Study)
- Live updates without refresh

### 🌍 Servers (Discord-like system)
- Internal servers inside the platform
- No external links needed
- Each server has independent posts
